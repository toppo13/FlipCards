# FlipCards - JavaScript

## 概要
HTML・CSS・Vanilla JavaScriptのみで実装した神経衰弱（カードめくり）ゲームです。

フレームワークを使用せず、DOM操作および状態管理の理解を目的として制作しました。

---

## 主な機能

・カードのランダム配置  
・クリックによるカード反転処理  
・一致判定ロジック  
・不一致時の自動裏返し処理  
・クリア判定  
・ゲーム状態の管理  

---

## 使用技術

- HTML
- CSS
- JavaScript（ES6）
- DOM操作
- イベントリスナー
- setTimeout / setInterval

---

## 実装内容

### シャッフル処理
Fisher–Yatesアルゴリズムを用いてカード配列をランダムに並び替えています。
外部ライブラリには依存せず、自前で実装しています。

---

### 状態管理

以下の状態を変数で明確に管理しています。

・1枚目に選択されたカード  
・2枚目のカード  
・入力ロック状態  
・ゲーム進行状況  

特に非同期処理中の多重クリックを防ぐため、
待機状態を管理する設計としています。

---

### 一致判定ロジック

2枚目が選択されたタイミングでカードの値を比較し、
一致・不一致を判定しています。

ロジックとUI操作を分離し、処理の流れが追いやすい構造を意識しました。

---

### 非同期制御

不一致時には `setTimeout` を使用し、
一定時間表示後にカードを裏返す処理を実装しています。

待機中は入力を制御し、状態破綻を防ぐ設計としています。

---

## 工夫した点

・連続クリックによるバグを防ぐ入力制御  
・ゲーム進行状態の明確な管理  
・非同期処理と状態管理の整合性維持  
・可読性を意識した関数分割  

単に動作するだけでなく、
状態が破綻しない設計を意識しました。
